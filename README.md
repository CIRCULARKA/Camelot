## Supported OS: Linux, macOS, Windows

## Build

Dotnet Core SDK 3.1 is required.

```
git clone https://github.com/ingvar1995/Camelot.git
cd Camelot
dotnet restore
dotnet build
dotnet run --project src/Camelot/Camelot.csproj
```
